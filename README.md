# Tic Tac Toe Game ❌⭕ – Java Console Version

This is a simple command-line **Tic Tac Toe** game built in Java, following clean coding and object-oriented principles.

---

## 📌 Features

- 2 Player Game (X and O)
- Turn-based input
- Win/tie detection
- Input validation
- Clean board display
